Compact Sets in Higher Dimensions
§1. What is Compactness?

In Chapter 3, we learned that compact sets in ℝ are exactly the closed and bounded sets (Heine–Borel Theorem). This same characterization holds in ℝⁿ, but compactness is even more powerful here because it imposes stronger control on infinite processes.

Core idea: Compact sets are "small enough" that infinite processes can be controlled.
Definition (Compactness via Sequences): A set K ⊆ ℝⁿ is compact if every sequence (xₖ) in K has a subsequence (x_{k_j}) that converges to a limit in K: $$(x_k) \subseteq K \implies \exists \text{ subsequence } (x_{k_j}) \text{ with } x_{k_j} \to L \in K.$$
Definition (Bounded Set in ℝⁿ): A set A ⊆ ℝⁿ is bounded if there exists M > 0 such that $$\|x\| \leq M \text{ for all } x \in A.$$
Theorem (Heine–Borel in ℝⁿ): A set K ⊆ ℝⁿ is compact if and only if K is closed and bounded.

This is the main computational tool: To check if a set is compact in ℝⁿ, just verify closure and boundedness!

§3. Key Properties of Compact Sets
Theorem (Closed Subset of Compact = Compact): If K is compact and F ⊆ K is closed, then F is compact.
Remark: This is extremely useful! To show a set is compact, we can show it's closed within a compact set and then appeal to this theorem.
Theorem (Nested Compact Sets): If K₁ ⊇ K₂ ⊇ K₃ ⊇ ⋯ is a nested sequence of nonempty compact sets in ℝⁿ, then $$\bigcap_{k=1}^{\infty} K_k \neq \emptyset.$$

§4. Continuous Functions on Compact Sets
Theorem (Extreme Value Theorem in ℝⁿ): If f : K → ℝ is continuous and K ⊆ ℝⁿ is compact, then f attains its maximum and minimum on K: $$\exists x_{\max}, x_{\min} \in K : f(x_{\min}) \leq f(x) \leq f(x_{\max}) \text{ for all } x \in K.$$

Theorem (Continuous Image of Compact = Compact): If f : K → ℝⁿ is continuous and K is compact, then f(K) is compact.
Intuition: Compactness is "preserved" by continuous functions. The image of a compact set stays compact.
§5. Open Covers and Heine–Borel
Definition (Open Cover): An open cover of K ⊆ ℝⁿ is a collection \{Oᵢ : i ∈ I\} of open sets such that $$K \subseteq \bigcup_{i \in I} O_i.$$ A finite subcover is a finite subcollection that still covers K.
Theorem (Heine–Borel: Finite Cover Characterization): A set K ⊆ ℝⁿ is compact if and only if every open cover of K has a finite subcover.
Remark: This is the topological definition of compactness. It's powerful because it works in general metric spaces and topological spaces, not just in ℝⁿ.
