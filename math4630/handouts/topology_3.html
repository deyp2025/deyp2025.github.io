Perfect and Connected Sets in Higher Dimensions
§1. Perfect Sets

A perfect set is a closed set with no isolated points. In ℝⁿ, perfect sets exhibit rich structure and can be highly nontrivial (fractals, Cantor-like sets, etc.).

Definition (Isolated Point): A point x ∈ A is isolated in A if there exists ε > 0 such that $$V_\varepsilon(x) \cap A = \{x\}.$$ Equivalently, x is not a limit point of A.
Definition (Perfect Set): A set P ⊆ ℝⁿ is perfect if (i) P is closed, and (ii) every point of P is a limit point of P (equivalently, P has no isolated points).
In words: Perfect sets are closed and "every point has neighbors in the set."

Theorem (Perfect Sets are Uncountable): Every nonempty perfect set in ℝⁿ is uncountable.
Remark: This is a deep result. Perfect sets may look complicated (fractals, Cantor-type constructions), but they always contain uncountably many points. Finite and countable sets cannot be perfect.
§2. Connected Sets

A connected set is one that is "all in one piece"—it cannot be split into two separated open parts.

Definition (Separated Sets): Two nonempty sets A, B ⊆ ℝⁿ are separated if $$\overline{A} \cap B = \emptyset \quad\text{and}\quad A \cap \overline{B} = \emptyset.$$
Definition (Disconnected and Connected): A set E ⊆ ℝⁿ is disconnected if it can be written as E = A ∪ B for some nonempty separated sets A and B. Otherwise, E is connected.
Intuition: Connected sets cannot have a "gap" in them.

§3. Path-Connected Sets

A stronger notion of connectedness: a set is path-connected if any two points can be joined by a continuous path.

Definition (Path-Connected): A set E ⊆ ℝⁿ is path-connected if for every x, y ∈ E, there exists a continuous function γ : [0,1] → E (a path) such that γ(0) = x and γ(1) = y.
Key fact: Path-connected $\implies$ connected. But the converse is false in ℝⁿ for n ≥ 2!

Theorem (Continuous Image of Connected = Connected): If f : E → ℝⁿ is continuous and E is connected, then f(E) is connected.
Theorem (Intermediate Value Property): If f : E → ℝ is continuous and E ⊆ ℝⁿ is connected, then f(E) is a connected subset of ℝ, i.e., an interval. Therefore, f attains all intermediate values between any two points in E.

Summary: Connected sets, compact sets, and continuous functions interact beautifully:
  • Connected + Continuous = Connected image
  • Compact + Continuous = Compact image (and extrema attained)
  • Path-connected is "nicer" but rarer than connected.
