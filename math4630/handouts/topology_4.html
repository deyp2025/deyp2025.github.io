Continuity and Uniform Continuity in Higher Dimensions
§1. Continuity at a Point

Continuity generalizes from ℝ to ℝⁿ in the obvious way: instead of the absolute value, we use the Euclidean norm. The ε-δ definition remains the same in spirit.

Definition (Continuity at a Point in ℝⁿ): Let f : D → ℝⁿ where D ⊆ ℝⁿ, and let x₀ ∈ D. We say f is continuous at x₀ if for every ε > 0, there exists δ > 0 such that $$x \in D \text{ and } \|x - x_0\| < \delta \implies \|f(x) - f(x_0)\| < \varepsilon.$$
Equivalently (sequential definition): f is continuous at x₀ if whenever xₖ → x₀, we have f(xₖ) → f(x₀).
Definition (Continuity on a Set): f is continuous on a set D if it is continuous at every point of D.

Theorem (Composition of Continuous Functions): If f : ℝⁿ → ℝᵐ is continuous at x₀ and g : ℝᵐ → ℝᵏ is continuous at f(x₀), then g ∘ f : ℝⁿ → ℝᵏ is continuous at x₀.
Theorem (Continuity via Open Sets): f : ℝⁿ → ℝᵐ is continuous if and only if the preimage of every open set is open: for all open U ⊆ ℝᵐ, $$f^{-1}(U) = \{x \in \mathbb{R}^n : f(x) \in U\} \text{ is open in } \mathbb{R}^n.$$
Remark: This topological characterization is powerful: continuous functions preserve openness (and closedness via complements).

Theorem (Extreme Value Theorem in ℝⁿ): If f : K → ℝ is continuous and K ⊆ ℝⁿ is compact, then f attains its maximum and minimum on K.

§3. Uniform Continuity

Continuity is pointwise: at each point x₀, the δ can depend on both ε and the location of x₀. Sometimes we need uniform control: a single δ works for all points.

Definition (Uniform Continuity): A function f : D → ℝⁿ (where D ⊆ ℝⁿ) is uniformly continuous if for every ε > 0, there exists δ > 0 such that for all x, y ∈ D: $$\|x - y\| < \delta \implies \|f(x) - f(y)\| < \varepsilon.$$
Key difference from continuity: The δ does not depend on the location (unlike pointwise continuity).
Uniform Continuity (ε-δ form):
For all ε > 0, there exists δ > 0 such that
∥x − y∥ < δ ⟹ ∥f(x) − f(y)∥ < ε
(This δ works everywhere on D simultaneously.)

Theorem (Heine–Cantor): If f : K → ℝⁿ is continuous and K ⊆ ℝⁿ is compact, then f is uniformly continuous on K.
Intuition: Compactness is strong enough to ensure uniform control over the continuity modulus.
§4. Lipschitz Continuity

An even stronger form of uniformity: the function doesn't change "too fast" anywhere.

Definition (Lipschitz Continuous): A function f : D → ℝⁿ is Lipschitz continuous with constant L > 0 if for all x, y ∈ D: $$\|f(x) - f(y)\| \leq L \|x - y\|.$$

Hierarchy: Lipschitz $\implies$ uniformly continuous $\implies$ continuous. But the reverse implications do not hold. Lipschitz is the "nicest" form of continuity with explicit control on the rate of change.
Theorem (Differentiability implies Lipschitz locally): If f : U → ℝⁿ is continuously differentiable on an open set U ⊆ ℝⁿ with bounded derivative (i.e., ∥Df(x)∥ ≤ M for all x ∈ U), then f is Lipschitz continuous with constant L = M.
§5. Summary: Continuity in ℝⁿ
Three forms of continuity (from weakest to strongest):
  1. Continuous: For each point and each ε, find δ (may depend on location)
  2. Uniformly continuous: For each ε, find δ (same δ works everywhere)
  3. Lipschitz continuous: Explicit bound on rate of change: ∥f(x) − f(y)∥ ≤ L∥x − y∥
Key theorems:
  • Continuity is preserved by composition and algebraic operations.
  • Continuous on compact $\implies$ uniformly continuous (Heine–Cantor).
  • Continuous on compact $\implies$ attains max/min (Extreme Value Theorem).
  • Continuous on connected $\implies$ image is connected (intermediate values).
