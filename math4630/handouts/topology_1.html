Topology in ℝⁿ — Handout 1

Open and Closed Sets in Higher Dimensions
§1. Moving from ℝ to ℝⁿ

In Chapter 3, we developed the topology of ℝ: open sets, closed sets, limit points, compactness, connectedness. All these concepts generalize beautifully to ℝⁿ—we just need to replace the absolute value with the Euclidean norm.

Definition (Euclidean Norm in ℝⁿ): For x = (x₁, x₂, …, xₙ) ∈ ℝⁿ, the Euclidean norm is $$\|x\| = \sqrt{x_1^2 + x_2^2 + \cdots + x_n^2}.$$ The distance between x, y ∈ ℝⁿ is $$d(x,y) = \|x - y\|.$$

Remark: When n=1, we recover d(x,y) = |x-y|.

Key insight: Distance in ℝⁿ works exactly like distance in ℝ—just computed differently.
§2. Neighborhoods and Open Sets
Definition (Neighborhood): Let x₀ ∈ ℝⁿ and ε > 0. The open ball (or ε-neighborhood) centered at x₀ is $$V_\varepsilon(x_0) = \{ x \in \mathbb{R}^n : \|x - x_0\| < \varepsilon \}.$$

In ℝ: V_ε(x₀) = (x₀ - ε, x₀ + ε) is an interval.

In ℝ²: V_ε(x₀) is an open disk.

In ℝ³: V_ε(x₀) is an open ball (sphere interior).

Definition (Open Set in ℝⁿ): A set O ⊆ ℝⁿ is open if for every x ∈ O, there exists ε > 0 such that $$V_\varepsilon(x) \subseteq O.$$

In words: A set is open if every point has an entire (small) ball around it that stays inside the set.

Theorem (Properties of Open Sets in ℝⁿ):
  1. The union of any collection of open sets is open.
  2. The intersection of finitely many open sets is open.
  3. ℝⁿ and ∅ are open.
§3. Closed Sets and Limit Points
Definition (Limit Point in ℝⁿ): A point x ∈ ℝⁿ is a limit point of a set A ⊆ ℝⁿ if every neighborhood V_ε(x) contains a point of A different from x: $$(V_\varepsilon(x) \setminus \{x\}) \cap A \neq \emptyset \text{ for all } \varepsilon > 0.$$
Theorem (Characterization via Sequences): A point x is a limit point of A if and only if there exists a sequence (aₖ) with aₖ ∈ A, aₖ ≠ x, such that aₖ → x.
Definition (Closed Set in ℝⁿ): A set F ⊆ ℝⁿ is closed if it contains all of its limit points.

Theorem (Complements Link Open and Closed): A set F ⊆ ℝⁿ is closed if and only if its complement Fᶜ = ℝⁿ ∖ F is open.
Theorem (Properties of Closed Sets in ℝⁿ):
  1. ℝⁿ and ∅ are closed.
  2. The intersection of any collection of closed sets is closed.
  3. The union of finitely many closed sets is closed.
§4. Closure and Interior
Definition (Closure): The closure of A ⊆ ℝⁿ, denoted Ā, is $$\overline{A} = A \cup L(A),$$ where L(A) is the set of all limit points of A.

Key fact: Ā is always closed, and is the smallest closed set containing A.

Definition (Interior): The interior of A ⊆ ℝⁿ, denoted A°, is the set of all points x ∈ A for which there exists ε > 0 such that V_ε(x) ⊆ A.

Key fact: A° is always open, and is the largest open set contained in A.
